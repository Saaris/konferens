18/10 we have the following seminaries:

Modern JavaScript Frameworks: React, Vue och Svelte En jämförelse av de mest populära ramverken för frontend: React, Vue och Svelte. För- och nackdelar med varje ramverk beroende på projektstorlek och behov. Praktiska exempel på hur man bygger en enkel applikation i varje ramverk. Diskussion om komponentbaserad arkitektur och hur man kan optimera prestanda.

Mastering Responsive Web Design Deep dive into media queries, fluid grids, and flexible images. Learn about mobile-first design principles. Explore CSS Grid and Flexbox to create modern, responsive layouts.

Introduction to TypeScript for Frontend Developers Learn the basics of TypeScript and how it integrates with JavaScript frameworks like React and Vue. Advantages of using TypeScript for larger projects and how to enforce type safety. How to set up TypeScript in a modern frontend project.

19/10

Web Performance Optimization: Techniques and Tools Strategies for improving page load times, such as code-splitting, lazy loading, and reducing render-blocking resources. Use tools like Lighthouse, Web Vitals, and Chrome DevTools to measure and optimize performance. Learn about the importance of caching and image optimization.

Building Accessible Websites: Best Practices Introduction to WCAG guidelines and why accessibility matters. Techniques for ensuring keyboard navigation, screen reader compatibility, and appropriate color contrast. Testing accessibility using automated tools and manual audits.

Component-Driven Development with Storybook Learn how to build, test, and document frontend components in isolation with Storybook. Explore component-driven development (CDD) principles and how they improve code reuse and collaboration. Integrate Storybook with frameworks like React, Vue, and Angular.
