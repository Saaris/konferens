For application we need you to sign up, see page below.
18/10 we have the following seminaries:

Modern JavaScript Frameworks-React, Vue och Svelte
A comparison of the most popular frontend frameworks: React, Vue, and Svelte.
Pros and cons of each framework depending on project size and needs.
Practical examples of how to build a simple application in each framework.

Mastering Responsive Web Design
Deep dive into media queries, fluid grids, and flexible images.
Learn about mobile-first design principles.
Explore CSS Grid and Flexbox to create modern, responsive layouts.

Introduction to TypeScript for Frontend Developers
Learn the basics of TypeScript and how it integrates
with JavaScript frameworks like React and Vue.

Testing Frontend Applications with Jest and Cypress
Introduction to automated testing for frontend applications.
Learn about unit testing with Jest and end-to-end testing with Cypress.

19/10 we have the following seminars:

Web Performance Optimization-Techniques and Tools
Strategies for improving page load times, such as code-splitting,
lazy loading, and reducing render-blocking resources
learn about the importance of caching and image optimization.

Building Accessible Websites-Best Practices
Introduction to WCAG guidelines and why accessibility matters.

Component-Driven Development with Storybook
Learn how to build, test, and document frontend components in isolation with Storybook.
Explore component-driven development (CDD) principles
and how they improve code reuse and collaboration.
Integrate Storybook with frameworks like React, Vue, and Angular.

20/10 we have the following seminars:

Server-Side Rendering with Next.js
Introduction to Next.js and its capabilities for server-side rendering (SSR).
Learn how SSR can improve SEO and initial load performance.
Explore dynamic routing, static site generation (SSG), and API routes with Next.js.

CSS-in-JS: Styled Components and Emotion
Overview of CSS-in-JS libraries like Styled Components and Emotion.
Benefits of component-level styling and how to manage dynamic styling in JavaScript.

Introduction to WebAssembly for Frontend Developers
Discover how WebAssembly can be used to improve performance in frontend applications.
Learn how to integrate WebAssembly with JavaScript.
